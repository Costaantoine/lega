# Documentation LEGA — Base RAG

Structure des dossiers :
- machines/       — Fiches techniques par type de machine (pelleteuse, chargeuse, grue...)
- constructeurs/  — Données constructeurs (CAT, Volvo, Komatsu, JCB, Liebherr...)
- transport/      — Guides transport (gabarit, démarches, tarifs bateaux...)
- reglementation/ — Réglementation (homologation Portugal, normes CE...)
- prix/           — Historique et références de prix du marché

Format recommandé : Markdown (.md) ou TXT (.txt)
L'agent Documentation indexe automatiquement ces fichiers lors du démarrage.
