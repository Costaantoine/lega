# Pelleteuse (Pelle hydraulique)

## Types principaux
- Mini-pelleteuse : 1-6T — urbain, jardins, petits travaux
- Pelleteuse standard : 8-20T — chantiers polyvalents (segment cœur LEGA)
- Grande pelleteuse : 20-50T+ — grands chantiers, carrières

## Marques principales sur le marché FR/PT
- **Caterpillar (CAT)** : 308, 320, 323, 336 — très demandé au Portugal
- **Volvo** : EC140, EC220, EC300 — bon rapport qualité/prix occasion
- **Komatsu** : PC130, PC210, PC290 — fiable, pièces disponibles
- **JCB** : JS130, JS220 — présence forte en France
- **Liebherr** : R926, R936 — haut de gamme
- **Hitachi** : ZX130, ZX210 — populaire au Portugal

- Heures moteur (idéal < 8000h)
- État des chenilles (remplacement = 8 000-15 000€)
